# Guard Panel API

Бэкенд веб-приложения панели управления задачами службы безопасности.

Проект выполнен в рамках выпускной квалификационной работы по теме:
**«Разработка веб-приложения панели управления задачами службы безопасности»**.

## Production

Сервис развернут и доступен по адресу:  
[https://guard-panel-api-production.up.railway.app/](https://guard-panel-api-production.up.railway.app/)

## Назначение

`guard-panel-api` предоставляет API для управления задачами, связанными с работой службы безопасности.
